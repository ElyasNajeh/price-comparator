## Project Description

This project is a Product Price Comparison Web Application that allows users to search for products across multiple online marketplaces and compare their prices in one place.

The application integrates multiple public APIs to retrieve real-time product data from different platforms such as:

- Amazon API
- AliExpress API
- eBay/Dummy Products API
- Currency Conversion API
- Text Correction / Spell Check API

The system helps users quickly compare products, prices, ratings, and product information from different online stores through a clean and user-friendly interface.
